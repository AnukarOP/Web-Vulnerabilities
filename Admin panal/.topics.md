## Synonyms

Admin panel bypass, admin login bypass, management console bypass, dashboard bypass, default credentials, weak credentials, SQL injection auth bypass, NoSQL injection auth bypass, response tampering, response manipulation, status code swap, parser confusion, parameter pollution, hidden admin endpoints.

## Attack patterns covered

Default credentials (`admin:admin`, `root:root`, `tomcat:tomcat`, `manager:manager`, `admin:password`, `admin:changeme`), SQL injection in username / password (`' OR '1'='1`, `admin' --`), XSS in login form, response manipulation (`200`/`302`, `success`/`true`), brute force without rate limit, directory fuzzing with wordlists, parameter removal (delete the `password` field), JavaScript file inspection for hardcoded credentials, HTML comment inspection, PHP type juggling (`==` vs `===`), Node.js parser confusion (`mysql.createConnection` `stringifyObjects`), `Content-Type: application/json` switch.
