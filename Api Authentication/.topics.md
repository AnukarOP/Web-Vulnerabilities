## Synonyms

## Attack patterns covered

Type juggling (`true`, `1`, `[]`, `{}`, `null`), array smuggling (`["admin"]`), nested-object operator injection (`{"$ne": null}` MongoDB, `{"$gt": ""}`), prototype pollution (`__proto__`, `constructor.prototype`), key duplication / case variants (`Password`, `PASSWORD`), HPP (`?username=a&username=b`), credential stuffing payloads, CRLF in values, null byte in values, oversized JSON, deeply nested JSON for DoS, JSON Schema validation bypass, content-type confusion (`application/json` vs `application/x-www-form-urlencoded`), boolean as string (`"true"`).
