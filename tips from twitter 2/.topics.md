## Topics covered

WAF XSS bypass (avoid `<script>`, exotic event handlers like `ontoggle`, `onpointerover`, `onanimationend`, HTML entity encoding, mixed-case tags, template literals, comment injection, keyword obfuscation via `\u00XX`), AEM dispatcher filter bypass (`/path.json/a.css`, `;.css`, `%20.css`, `#.css`, `?.css`), file-upload extension tests for PHP / ASP / JSP, SQL injection in `sitemap.xml` and variants (`sitemap_index.xml`, `post-sitemap.xml`, `category-sitemap.xml`), SSRF via PDF generation through headless browsers (`<iframe src="http://169.254.169.254/latest/meta-data/">`), blind SSRF via webhook URLs (AWS / GCP / Azure metadata endpoints, `localhost`, loopback IPv6, `0.0.0.0`), cache deception via appending static extensions to sensitive endpoints (`profile.css`, `api-key.png`), bug-bounty mindset (chain low + low + low = critical).
